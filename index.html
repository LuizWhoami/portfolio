Desafio Web: SQL Injection

Fácil 20/10/2023

Descrição do Desafio

O desafio consiste em explorar uma vulnerabilidade de SQL Injection em um sistema de login para obter acesso à conta do administrador.

Solução

1. Identifiquei que o campo de login estava vulnerável a SQL Injection através de testes básicos com aspas simples.

2. Utilizei a seguinte payload para bypassar a autenticação:

admin' --

3. Isso resultou na seguinte query sendo executada no banco de dados:

SELECT * FROM users WHERE username = 'admin' -- ' AND password = 'qualquer'

4. O comentário (--) fez com que a verificação da senha fosse ignorada, permitindo o login como admin.

Flag Capturada

HTB{sql1_3asy_p34sy}

Prevenção

Para prevenir esse tipo de ataque, recomendo:

  • Usar prepared statements
  • Validar e sanitizar todas as entradas
  • Implementar WAF (Web Application Firewall)
