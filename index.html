CTF Challenges

Basic Pentesting - TryHackMe

TryHackMe
Fácil Web Security

Descrição

Este desafio envolve a exploração de vulnerabilidades em um servidor web mal configurado, incluindo enumeração de diretórios, brute force e exploração de serviços desatualizados.

Passos da Solução

1. Enumeração inicial com nmap:

nmap -sV -sC -oA scan 10.10.10.10

2. Identificação do serviço SMB exposto:

smbclient -L //10.10.10.10

3. Exploração do diretório compartilhado:

smbclient //10.10.10.10/anonymous

Flags Capturadas

THM{basic_pentesting_done}

Aprendizados

  • Importância de desativar compartilhamentos anônimos
  • Necessidade de manter serviços atualizados
  • Técnicas básicas de enumeração
