TeleAlert — Bot de Monitoramento

Bot Telegram que monitora endpoints HTTP, detecta mudanças de status e envia alertas com latência, código de resposta e diff de conteúdo. Ideal para detectar mudanças suspeitas em aplicações monitoradas durante recon passivo.

Python python-telegram-bot aiohttp asyncio

LogHunter — Análise de Logs SSH

Script Python que analisa logs do sistema (/var/log/auth.log) em busca de tentativas de brute-force, IPs suspeitos e padrões de ataque. Gera relatório em JSON com geolocalização de IPs e score de risco.

Python Regex GeoIP2 JSON

SecDash — Dashboard de Vulnerabilidades

Interface web para visualização e triagem de vulnerabilidades encontradas em scans. Integra com saída do Nmap (XML) e exibe hosts, portas abertas, serviços e CVEs relacionados com filtros interativos. 100% vanilla JS.

HTML/CSS JavaScript Nmap XML NVD API

Ferramenta de reconhecimento que orquestra Nmap, Gobuster, Whatweb e Nikto em sequência lógica, salva output estruturado por alvo e gera sumário automático com surface de ataque identificada.

Python subprocess Nmap Gobuster
